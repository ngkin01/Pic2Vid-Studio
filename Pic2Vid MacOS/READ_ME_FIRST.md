# 🎬 Pic2Vid Studio — Hướng dẫn cài đặt (MacBook)

Ứng dụng này giúp bạn tự động:
1. Đưa ảnh sản phẩm vào **Gemini** → tạo ảnh đẹp hơn
2. Từ ảnh đó đưa vào **Meta AI** → tạo video TikTok

---

## 📋 Yêu cầu trước khi bắt đầu

- MacBook đang dùng macOS
- Có tài khoản **Google** (để dùng Gemini)
- Có tài khoản **Facebook** (để dùng Meta AI)
- Có **Google Chrome** cài sẵn trên máy

---

## 🚀 Cài đặt lần đầu (chỉ làm 1 lần)

### Bước 1 — Cài Node.js

1. Mở trình duyệt, vào: **https://nodejs.org**
2. Bấm nút **"Download Node.js (LTS)"** — nút màu xanh lớn
3. Mở file vừa tải về → bấm **Continue** → **Continue** → **Install**
4. Nhập mật khẩu MacBook nếu được hỏi → **Install Software**
5. Chờ cài xong → bấm **Close**

---

### Bước 2 — Giải nén folder Product Studio

1. File bạn nhận được tên là `pic2vid-mac.zip`
2. Double-click vào file đó để giải nén
3. Bạn sẽ thấy folder **pic2vid-mac** — kéo nó vào **Desktop** cho dễ tìm

---

### Bước 3 — Chạy Setup lần đầu

> ⚠️ Bước này chỉ làm **1 lần duy nhất**

1. Mở folder **pic2vid-mac** trên Desktop
2. **Double-click** vào file **`SETUP.command`**
3. Mac hỏi *"Are you sure you want to open it?"* → bấm **Open**
4. Terminal tự mở và chạy, chờ cài xong (khoảng 2-3 phút)

---

### Bước 4 — Đăng nhập Gemini

Sau khi cài xong, Terminal sẽ tự mở Chrome và vào trang Gemini:

1. Đăng nhập bằng tài khoản **Google** của bạn như bình thường
2. Đăng nhập xong, thấy trang chat Gemini → **quay lại Terminal**
3. Nhấn **Enter** trong Terminal

---

### Bước 5 — Đăng nhập Meta AI

Ngay sau đó Chrome sẽ mở trang Meta AI:

1. Đăng nhập bằng tài khoản **Facebook** của bạn
2. Đăng nhập xong, thấy trang chat Meta AI → **quay lại Terminal**
3. Nhấn **Enter** trong Terminal

---

### ✅ Setup hoàn tất!

Terminal sẽ hiện thông báo *"Setup hoàn tất"*. Từ giờ bạn không cần làm lại bước này nữa.

---

## 📱 Sử dụng hàng ngày

### Mỗi lần muốn dùng:

1. Mở folder **pic2vid-mac** trên Desktop
2. **Double-click** vào file **`START.command`**
3. Chờ vài giây → trình duyệt tự mở trang **Product Studio**

> 💡 Giữ cửa sổ Terminal mở trong khi dùng. Đóng Terminal là ứng dụng tắt.
