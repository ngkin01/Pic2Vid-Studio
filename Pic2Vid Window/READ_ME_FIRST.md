# 🎬 Pic2Vid Studio — Hướng dẫn cài đặt (Windows)

Ứng dụng này giúp bạn tự động:
1. Đưa ảnh sản phẩm vào **Gemini** → tạo ảnh đẹp hơn
2. Từ ảnh đó đưa vào **Meta AI** → tạo video TikTok

---

## 📋 Yêu cầu trước khi bắt đầu

- Máy tính Windows 10 hoặc 11
- Có tài khoản **Google** (để dùng Gemini)
- Có tài khoản **Facebook** (để dùng Meta AI)
- Có **Google Chrome** cài sẵn trên máy

---

## 🚀 Cài đặt lần đầu (chỉ làm 1 lần)

### Bước 1 — Cài Node.js

1. Mở trình duyệt, vào: **https://nodejs.org**
2. Bấm nút **"Download Node.js (LTS)"** — nút màu xanh lớn
3. Mở file vừa tải về → bấm **Next** liên tục → **Install**
4. Bấm **Yes** nếu Windows hỏi cho phép cài
5. Chờ cài xong → bấm **Finish**

---

### Bước 2 — Giải nén folder Product Studio

1. File bạn nhận được tên là `pic2vid-win.zip`
2. Chuột phải vào file → chọn **Extract All** → **Extract**
3. Bạn sẽ thấy folder **pic2vid-win** — kéo nó vào **Desktop** cho dễ tìm

---

### Bước 3 — Chạy Setup lần đầu

> ⚠️ Bước này chỉ làm **1 lần duy nhất**

1. Mở folder **pic2vid-win** trên Desktop
2. Chuột phải vào file **`SETUP.bat`** → chọn **Run as administrator**
3. Bấm **Yes** nếu Windows hỏi
4. Cửa sổ đen xuất hiện, chờ nó cài xong (khoảng 2-3 phút)

---

### Bước 4 — Đăng nhập Gemini

Sau khi cài xong, cửa sổ sẽ tự mở Chrome và vào trang Gemini:

1. Đăng nhập bằng tài khoản **Google** của bạn như bình thường
2. Đăng nhập xong, thấy trang chat Gemini → **quay lại cửa sổ đen**
3. Nhấn **Enter**

---

### Bước 5 — Đăng nhập Meta AI

Ngay sau đó Chrome sẽ mở trang Meta AI:

1. Đăng nhập bằng tài khoản **Facebook** của bạn
2. Đăng nhập xong, thấy trang chat Meta AI → **quay lại cửa sổ đen**
3. Nhấn **Enter**

---

### ✅ Setup hoàn tất!

Cửa sổ sẽ hiện thông báo *"Setup hoàn tất"*. Từ giờ không cần làm lại bước này nữa.

---

## 🖥️ Tạo shortcut ngoài Desktop (tiện dùng hàng ngày)

1. Mở folder **pic2vid-win**
2. Chuột phải vào **`START.bat`** → chọn **Create shortcut**
3. Kéo file **`START - Shortcut.bat`** vừa tạo ra ngoài Desktop

Từ giờ double-click shortcut đó ngoài Desktop là chạy được luôn.

---

## 📱 Sử dụng hàng ngày

### Mỗi lần muốn dùng:

1. Double-click **shortcut START.bat** ngoài Desktop (hoặc file START.bat trong folder)
2. Cửa sổ đen xuất hiện + trình duyệt tự mở trang **Product Studio**

> 💡 Giữ cửa sổ đen mở trong khi dùng. Đóng cửa sổ đen là ứng dụng tắt.
